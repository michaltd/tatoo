Um Tatoo zu kompilieren müssen die externen Libarys unter tatoo.extlib in den Buildpaht eingebunden
werden. Unter Exlipse kann das geschehen indem die Projektproperties geöffnet werden:
Menü "Project" -> Properties -> Java Buildpath
und dort dann unter Libaries die externen Libaries mit "Add External JARs" eingebunden werden.
