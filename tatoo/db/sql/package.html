Treiber für die SQL-Datenbank H2.

Für eine generelle Beschreibung der Schnittstelle zu einer Datenstruktur siehe auch Beschreibung unter {@link tatoo.db}

Neu in diesem Paket sind die Klassen welche von der Klasse {@link DBSchemaPattern} erben. Aber dazu später mehr.
In dem Paket tatoo.db.sql liegt eine Schema-Datei. Es handelt sich hier um eine XML-Datei welche die Klassen und deren Attribute mit Tabellen und Spalten verknüpft. Normalerweise würde man heutzutage mit Annotiationen arbeiten wenn eine Klasse Serialisiert werden soll, aber da in tatoo ein importieren von älteren Datenstrukturen erlaubt sein soll (Eine Armeeliste wurd auf Rechner A mit der tatoo-Version a gespeichert und soll nun auf Rechner B mit der tatoo-Version b eingelesen werden) ist es erforderlich deren Schema mit der Armeeliste zu speichern. Ansonsten müsste das kompilierte tatoo mit jeder Armeeliste mitgeführt werden :)
Zurück zur Klasse DBSchemaPattern. Das soeben beschriebene XML-Schema wird eingelesen und je nach XML-Element werden verschiedene Instanzen von Kindern von DBSchemaPattern instaniiert. Diese Bilden die Attribute und Klassen aus der Schema-Datei in einer Objekthirarchie ab.
