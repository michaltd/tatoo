Veränderbare Attribute für Entities.

Conditions werden in Entities als Attribute benutzt. Eine Condition stellt im Grunde genommen ein Attribut dar, welches von anderen Werten, Attributen oder anderen Entities abhängig sein kann. So lassen sich Attribute während der Laufzeit auf einfache Weise berechnen und ändern.

Ein Beispiel:
Eine Einheit B hängt von einer anderen Einheit A in der Form ab, dass für jede Einheit A eine Einheit B in den Armeeplan aufgenommen werden darf.

Einheit B bekommt nun für das count-Attribut eine {@link CalculatedNumberCondition} übergeben welche wiederum auf das Attribut count von Einheit A verweist. Ändert sich der Wert von A.count so ist von dieser Änderung automatisch der Wert von B.count betroffen.
